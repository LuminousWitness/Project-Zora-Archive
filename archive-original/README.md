# Archival Originals

This directory is reserved for unaltered copies of Project Zora material already made public by Christopher Anthony John Rimmer.

## Rules

- Preserve source files without modification whenever possible.
- Retain original filenames in metadata even when an archive-safe filename is also assigned.
- Record source URL, publication date, acquisition date, media type, and SHA-256 checksum.
- Do not overwrite originals with cropped, enhanced, compressed, translated, redacted, or reformatted versions.
- Store access copies and derived material separately.

Suggested subdirectories may be organized by year, platform, or media type once the first source collection is assessed.
