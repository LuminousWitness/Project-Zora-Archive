# Archive Index

This directory will contain the navigational and integrity records for the Project Zora Archive.

Planned files include:

- `chronological-index.csv` — one row per archived item in publication order;
- `source-manifest.csv` — filenames, provenance, checksums, and acquisition details;
- `topic-index.md` — human-readable navigation by subject;
- `relationships.csv` — links among posts, replies, attachments, transcripts, and later corrections;
- `mirrors.md` — independent archive locations and synchronization status.

Dates or relationships that cannot be established confidently must be marked uncertain rather than inferred silently.
